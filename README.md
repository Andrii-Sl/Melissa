# 🚗 AutoParts — Подбор и доставка автозапчастей из Европы

Современный веб-сервис для подбора автозапчастей по VIN-коду с личным кабинетом, системой заказов и онлайн-оплатой.

---

## 🔥 Возможности

- 🔎 Подбор запчастей по VIN
- 🚗 Распознавание автомобиля (BMW, Audi, VW и др.)
- 📦 Личный кабинет с заказами
- 💳 Онлайн-оплата (Stripe: Visa / MasterCard / Apple Pay)
- 📊 Статусы заказов:
  - В подборе
  - Ожидает оплаты
  - Доставлен
- 🧠 Расчет цены с маржой
- 📱 Адаптивный интерфейс (мобильный + десктоп)

---

## 🛠 Технологии

- Next.js 14 (App Router)
- React
- TypeScript
- Stripe API (оплата)
- CSS (кастомный UI)
